# Условия использования

Дата последнего обновления: июль 2026.

## Общие положения

Mineradio for Sade (включая интеграцию Discord Rich Presence "MineRadio") — некоммерческий десктопный проект с открытым исходным кодом, распространяемый по лицензии GPL-3.0. Используя приложение, вы соглашаетесь с настоящими условиями.

## Что делает приложение

- Локальный музыкальный плеер (MP3/FLAC) с визуальными эффектами.
- Опциональная интеграция с Яндекс Музыкой через вход в личный аккаунт пользователя (OAuth).
- Опциональная интеграция с Discord Rich Presence: при включении в статусе Discord отображается текущий трек (название, исполнитель, обложка). Приложение не имеет доступа к серверам, сообщениям или другим данным аккаунта Discord — используется только Rich Presence API для отображения статуса.

## Ограничения

- Приложение не предоставляет обход платной подписки Яндекс Музыки, взлом качества звука или инструменты для пересылки/распространения музыкального контента.
- Приложение не является официальным продуктом Яндекс Музыки или Discord и не аффилировано с ними.

## Ответственность
