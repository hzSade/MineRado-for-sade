# Приватность и пользовательские данные

Mineradio for Sade — локальное desktop-приложение. Состояние авторизации, cookie, история воспроизведения и поиска, пользовательские обложки и тексты песен не должны попадать в git-репозиторий.

## Локальные данные

Приложение может хранить на компьютере пользователя:

- данные авторизации в Яндекс Музыке (OAuth-токен)
- путь и индекс локальной музыкальной библиотеки (MP3/FLAC)
- историю поиска
- пользовательские обложки альбомов
- пользовательские тексты песен
- настройки положения и визуального оформления лирик-сцены
- локальный кэш анализа ритма
- кэш загруженных установочных пакетов обновлений

Эти данные нужны только для локальной работы приложения и не являются частью открытого репозитория.

## Что не должно попадать в репозиторий

- токены и cookie авторизации Яндекс Музыки
- `updates/`
- `node_modules/`
- собранные Electron-пакеты (`dist/`, `win-unpacked/`)
- локальные музыкальные файлы пользователя
- любые данные аккаунта, токены, QR-коды входа

## Сторонние сервисы

При входе через Яндекс Музыку пользователь обязан соблюдать условия использования этого сервиса. Mineradio for Sade не предоставляет способов обхода платной подписки, взлома качества аудио или пересылки/распространения музыкального контента третьим лицам.
