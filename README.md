# Mineradio for Sade

Mineradio for Sade — доработанная версия Windows-плеера Mineradio: погодное радио, лирик-сцена, частицы и 3D-полка плейлистов дополнены поддержкой **локальной библиотеки MP3/FLAC** и **Яндекс Музыки** вместо оригинальных Netease/QQ Music.

## Скачать установщик Windows

| Способ | Кому подходит | Ссылка |
| --- | --- | --- |
| GitHub Releases | Все пользователи | [MineradioSetup.exe](https://github.com/coffeedsd/MineRado-for-sade/releases/tag/MineRadio) |

Нужно скачать и запустить именно `Mineradio-1.2.3-Setup.exe`. Не скачивайте `Source code`, `.blockmap`, `latest.yml` и не используйте папку `win-unpacked` как установочный пакет.

## Если скачивание или установку блокирует

Малораспространённые Electron-приложения без цифровой подписи иногда помечаются браузером, Windows Defender или SmartScreen как потенциально опасные. Убедитесь, что файл скачан именно из GitHub Releases этого репозитория и называется `Mineradio-1.2.3-Setup.exe`.

1. Если браузер предупреждает при скачивании — откройте список загрузок, нажмите `...` рядом с файлом и выберите «Оставить» / «Всё равно скачать».
2. Если Windows SmartScreen показывает синее окно — нажмите «Подробнее», затем «Выполнить в любом случае».
3. Если антивирус явно определяет файл как троян и помещает в карантин — не запускайте его принудительно; удалите файл и скачайте заново из Releases. Если проблема повторяется — сообщите со скриншотом.

## Основные возможности

- Погодное радио на Open-Meteo — плейлист подбирается под текущую геолокацию, город и настроение погоды
- Главный экран с погодным радио, ежедневными рекомендациями, персональным радио, разделом «Продолжить слушать», музыкальным портретом и разделом «Мои плейлисты»
- Фоновая заставка-«галактика» на главном экране в состоянии покоя
- Визуальный режим в духе кинематографичной операторской работы, синхронизированный с ритмом трека
- Отдельный визуальный режим для длинных подкастов и DJ-сетов
- Лирик-сцена: собственные тексты песен, позиционирование и визуальные настройки
- Загрузка и обрезка собственных обложек альбомов
- 3D-полка плейлистов по правому клику мыши с прокруткой очереди
- **Локальная музыкальная библиотека**: сканирование папок с MP3/FLAC, локальные обложки и тексты `.lrc`/`.txt`
- **Интеграция с Яндекс Музыкой** вместо оригинальных Netease/QQ Music — вход через OAuth, поиск, плейлисты
- Значок в системном трее и сворачивание в трей
- Проверка обновлений и загрузка через GitHub Releases

## Как пользоваться

Скачайте установщик из раздела Releases и запустите `Mineradio-1.2.3-Setup.exe`. Устанавливается ярлык на рабочий стол.

Если вы обновляетесь со старой версии — рекомендуется сначала удалить старую версию, затем поставить новую «с чистого листа».

## Сборка из исходников

```bash
npm install
npm start
npm run build:win
```

Desktop-версия запускается через главный процесс Electron, который поднимает локальный сервер. `npm run build:win` собирает установщик NSIS для Windows, результат — в папке `dist/`.

## Механизм обновлений

Приложение проверяет последний релиз в GitHub Releases этого репозитория. Если удалённая версия новее локальной, в приложении появится предложение скачать и установить обновление.

Для локальной проверки механизма обновлений можно указать переменную окружения `MINERADIO_UPDATE_MANIFEST` с путём/адресом до JSON-манифеста, имитирующего Release.

## О сторонних музыкальных сервисах

Mineradio for Sade не является официальным клиентом Яндекс Музыки и не аффилирован с ней. Интеграция используется только для личного использования с собственной учётной записью пользователя в рамках пользовательского соглашения сервиса. Проект не предоставляет обхода платной подписки, взлома качества звука или пересылки/распространения музыкального контента.

## Данные пользователя и приватность

Cookies авторизации, история поиска, собственные обложки и тексты песен, кэш анализа ритма хранятся только в локальной пользовательской папке на компьютере и не должны попадать в репозиторий.
